Strategy Pattern
================================================================

전략 패턴(Strategy Pattern)은 실행 중에 알고리즘을 선택할 수 있게 하는 행위 소프트웨어 디자인 패턴
- 특정한 계열의 알고리즘들을 정의
- 각 알고리즘을 캡슐화
- 알고리즘들을 해당 계열 안에서 상호 교체가 가능하게 만듬
